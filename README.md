A beginner-friendly, hands-on Splunk project that teaches log ingestion, SPL queries, dashboards, and alerts using real-world security log analysis use cases.
Hands-On Splunk Log Analysis Project | SPL, Dashboards & Alerts

This repository provides a hands-on, beginner-friendly Splunk project designed to demonstrate core concepts of log ingestion, analysis, and visualization. It is ideal for aspiring cybersecurity professionals, SOC analysts, and system engineers looking to gain practical experience with real-world log data.

The project walks learners through ingesting structured logs, extracting meaningful fields, writing SPL queries, building dashboards, and configuring alerts. Each step reflects common security operations and monitoring use cases, helping bridge the gap between theory and practical SOC workflows.

Project Objectives

Teach Splunk log ingestion and indexing

Demonstrate essential SPL (Search Processing Language) commands

Extract and analyze useful fields from log data

Build dashboards and alerts for operational and security insights

Requirements

Splunk Enterprise (local install) or Splunk Cloud (free trial)

Downloaded practice log dataset: bns_practice.csv
